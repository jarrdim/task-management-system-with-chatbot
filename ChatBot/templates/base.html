Chatbot - Chirpy is designed to respond to customer inquiries regarding the bank and its services, such as account opening, loans, and various digital product.The methodology employed to develop this chatbot involves several key steps. Firstly, the chatbot leverages natural language processing tools, such as the Natural Language Toolkit (NLTK), to tokenize and stem words from a dataset of predefined intents stored in a JSON file. These intents consist of patterns associated with specific tags and responses. The data is then preprocessed to create a "bag of words" representation for training a neural network. The code utilizes the TensorFlow-based library, TFLearn, to construct a feed-forward neural network with an input layer, two hidden layers, and an output layer with softmax activation. This neural network is trained on the prepared data, with the training process visualized through metrics. The trained model is capable of predicting the intent tag based on user input.
